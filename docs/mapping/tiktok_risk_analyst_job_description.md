### Responsibilities

- Evaluate risks across risk domains using intelligence tools and create scalable solutions that allow users and businesses to interact in a trustworthy manner.
- Support the design and optimization of ad policies, moderation processes, scalable systems and tools to identify risks and drive efficiencies within integrity mechanisms.

## Qualifications

### Qualifications

Must have a Master's degree or foreign equivalent degree in Computer Science, Engineering (any), Data Science, Business Analytics, Business, Statistics, Mathematics, or a related field, and 1 year of related work experience; OR a Bachelor’s degree or foreign equivalent degree in Computer Science, Engineering (any), Data Science, Business Analytics, Business, Statistics, Mathematics, or a related field, and 3 years of related work experience.

Of the required experience, must have 1 year of experience in each of the following:

- Using SQL or Python to execute data analysis and create data visualizations;
- Leveraging Python and Python machine learning programming language libraries for risk insight data analysis;
- Performing statistical imputations to work with missing data, executing statistical analysis, and performing statistical-driven tests on models to define hypotheses;
- Using SQL for performance and operations metrics analysis and insight development, including extracting data from data lakes and adding key indicators and attributes to datasets; and
- Performing data blending and data wrangling to accumulate data from multiple sources to build dashboards.
